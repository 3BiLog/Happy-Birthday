# Happy Birthday

Ứng dụng Android đơn giản được xây dựng bằng Kotlin và Jetpack Compose.

## Giới thiệu

Đây là project luyện tập Android đầu tiên sử dụng Jetpack Compose để tạo giao diện chúc mừng sinh nhật.

Ứng dụng hiển thị:

* Lời chúc sinh nhật
* Tên người gửi
* Bố cục giao diện bằng Column
* Modifier để căn chỉnh và tạo khoảng cách

## Công nghệ sử dụng

* Kotlin
* Jetpack Compose
* Material 3
* Android Studio

## Kiến thức đã học

* `@Composable`
* `Text`
* `Column`
* `Modifier`
* `padding()`
* `align()`
* `fontSize`
* `Surface`
* `MaterialTheme`
